# Brain Weekly Review

Version: `1.0.0`

1. Inventory inbox and proposed records without moving anything.
2. Propose classifications, derived notes, tag changes, and source links.
3. Review every active project's priorities and next actions.
4. Surface conflicts and lessons that may change current views.
5. Preview all changes and route protected changes through the broker.
6. Create an archived dated weekly review and regenerate indexes.
