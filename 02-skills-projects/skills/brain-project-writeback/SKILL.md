# Brain Project Writeback

Version: `1.0.0`

1. Update the registered project's status, decisions, and next actions directly.
2. Cite the repository revision or artifact that supports the update.
3. Put broader lessons and Layer 1 proposals in the inbox.
4. Do not rewrite identity or strategy directly.
5. Regenerate project indexes after updates.
