# Brain Project Handoff

Version: `1.0.0`

1. Read the private project record and search only relevant Layer 1 context.
2. Draft outcomes, constraints, decisions, priorities, and cited sources.
3. Preview the exact brief and its private destination.
4. Obtain both approvals if the brief will leave private brain storage.
5. Store the dated authoritative brief in the private project record area.
6. Record this skill version in the brief.
