# Brain Capture

Version: `1.0.0`

1. Preserve the user's wording and distinguish it from agent interpretation.
2. Run `python -m main_brain capture --title "<title>" --body "<body>"`.
3. Add explicit source and project links.
4. Leave classification as `inbox`; weekly review determines durable placement.
5. Return the created private path without exposing its contents publicly.
